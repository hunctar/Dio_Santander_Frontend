Tags de cabeçalhos - h1 à h6

As tags de cabeçalhos são usadas para definir títulos e subtítulos em uma página HTML. Elas vão de H1 a H6, onde H1 é o mais importante e H6 é o menos importante. A forma de usar é:

<h1>Título Principal</h1>

As tags de cabeçalhos são importantes para a estruturação do conteúdo e para SEO (Search Engine Optimization). Elas ajudam os motores de busca a entenderem a hierarquia do conteúdo da página. Veja abaixo exemplos de uso:

Título Principal

Subtítulo 1

Subtítulo 2

Subtítulo 3

Subtítulo 4
Subtítulo 5

Tag de parágrafo - p

A tag P é usada para definir parágrafos em HTML. Ela é uma das tags mais comuns e é usada para agrupar texto em blocos. A forma de usar é:

<p>Este é um parágrafo.</p>

Tag Mark

A tag Mark é usada para destacar texto importante ou relevante. Ela é útil para chamar a atenção do usuário para informações específicas. A forma de usar é:

<mark>Texto destacado</mark>

Veja abaixo um exemplo de uso:
Este texto está destacado.

Voltar

Tag Small

A tag Small é usada para definir texto menor que o texto normal. Ela é útil para notas de rodapé, avisos ou informações adicionais. A forma de usar é:

<small>Este é um texto menor.</small>

Veja abaixo um exemplo de uso:
Este é um texto normal.
Este texto é menor que o texto normal.

Tag Itálico

A tag I é usada para mostrar texto em itálico. Ela é útil para destacar palavras ou frases que precisam de ênfase. A forma de usar é:

<i>Este texto é em itálico<i>

Veja abaixo um exemplo de uso:
Este texto está em itálico.

Tag Underscore

A tag u é usada para mostrar texto sublinhado. Ela é útil para destacar palavras ou frases que precisam de ênfase visual. A forma de usar é:

<u>Este texto é sublinhado</u>

Veja abaixo um exemplo de uso:
Este texto está sublinhado.

Tag Strong (Negrito)

A tag strong é usada para mostrar texto em negrito. Ela é útil para destacar palavras ou frases que precisam de ênfase. A forma de usar é:

<strong>Este texto é em negrito</strong>

Veja abaixo um exemplo de uso:
Este texto está em negrito.

Lista Ordenada

A tag ol é usada para criar listas ordenadas, onde os itens são numerados. A forma de usar é:

<ol><li>Item 1</li><li>Item 2</li></ol>

Veja abaixo um exemplo de uso:

  1. Item 1
  2. Item 2
  3. Item 3

Voltar

Lista não ordenada

A tag ul é usada para criar listas não ordenadas, onde os itens são marcados com pontos ou outros símbolos. A forma de usar é:

<ul><li>Item 1</li><li>Item 2</li></ul>

Veja abaixo um exemplo de uso:

  • Item 1
  • Item 2
  • Item 3

Tag a, utilizada para links

A tag a é usada para criar links para outras páginas ou recursos. Ela é essencial para a navegação na web. A forma de usar é:

<a href="URL">Texto do link</a>

Veja abaixo um exemplo de uso:
Este é um link para Example.com

Linha Horizontal

A tag hr é usada para criar uma linha horizontal, que pode ser usada para separar seções de conteúdo. Ela não tem conteúdo entre as tags, apenas é fechada com uma barra. A forma de usar é:

<hr>

Veja abaixo um exemplo de uso:

Esta é uma linha horizontal.

Escrita inferior

A tag sub é usada para escrever texto na parte inferior da linha, como em fórmulas químicas ou matemáticas. A forma de usar é:

<sub>Texto inferior</sub>

Veja abaixo um exemplo de uso:
H2O é a fórmula química da água.

Voltar

Escrita superior

A tag sup é usada para escrever texto na parte superior da linha, como em expoentes matemáticos. A forma de usar é:

<sup>Texto superior</sup>

Veja abaixo um exemplo de uso:
23 é igual a 8.

Blockquote

A tag blockquote é usada para citar longas passagens de texto de outras fontes. Ela geralmente é usada para destacar citações ou trechos importantes. A forma de usar é:

<blockquote>Texto da citação</blockquote>

Veja abaixo um exemplo de uso:

"A vida é o que acontece enquanto você está ocupado fazendo outros planos." - John Lennon

Voltar

Tag Font

A tag font é usada para definir a fonte, tamanho e cor do texto. Ela é menos comum hoje em dia, pois o CSS é preferido para estilização. A forma de usar é:

<font face="Arial" size="3" color="blue">Texto</font>

Veja abaixo um exemplo de uso:
Este texto está estilizado com a tag font.

Voltar

Tag Del

A tag del é usada para indicar que um texto foi removido ou excluído. Ela é útil para mostrar alterações em documentos ou textos revisados. A forma de usar é:

<del>Texto excluído</del>

Veja abaixo um exemplo de uso:
Este texto foi removido.

Voltar

Tag abbr

A tag abbr é usada para definir abreviações ou siglas. Ela é útil para fornecer explicações sobre termos abreviados. A forma de usar é:

<abbr title="Abreviação">Sigla</abbr>

Veja abaixo um exemplo de uso:
ONU é uma organização internacional.
