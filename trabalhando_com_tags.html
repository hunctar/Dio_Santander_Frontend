Aside - é como uma section, mas não é importante para a pagina.
Nav - é como uma section, mas é para links de navegação.
Article - é como uma section, mas é para artigos independentes.

Título do Artigo

Este é o conteúdo do artigo. Ele pode conter texto, imagens, vídeos, etc.

Você pode usar a tag strong para destacar texto importante.

Ou a tag em para enfatizar texto.

Você também pode usar a tag code para mostrar código.

Ou a tag mark para destacar texto.
